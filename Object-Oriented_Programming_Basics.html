Lesson Plan: Introduction to Object-Oriented Programming for 6th Graders

Objective: By the end of this lesson, students will understand the basic principles of object-oriented programming (OOP) and be able to apply these concepts through the creation of a simple Car class in Java.

1. Introduction to Object-Oriented Programming (15 minutes)

A. What is Object-Oriented Programming?

  • Definition: OOP is a way of organizing code to model real-world things using classes and objects.
  • Key Concepts: Classes, Objects, Inheritance, Polymorphism

B. Real-World Example:

  • Example Scenario: Imagine a car dealership. Each car has its own make, model, year, and color. Despite their differences, all cars share common characteristics and behaviors.

2. Key Concepts of OOP (20 minutes)

A. Classes and Objects:

  • Class: A blueprint for creating objects. It defines the properties (attributes) and behaviors (methods) of the objects.
    • Example: Car class
  • Object: An instance of a class. Each object has its own set of properties and can perform behaviors defined in the class.
    • Example: Specific car like a "Red 2020 Toyota Corolla"

B. Inheritance:

  • Inheritance: Allows a class to inherit properties and methods from another class.
    • Example: ElectricCar class that inherits from Car class

C. Polymorphism:

  • Polymorphism: Allows objects of different classes to be treated as objects of a common super class.
    • Example: Treating both Car and ElectricCar objects as Car type

3. Designing the 'Car' Class (30 minutes)

A. Defining the Class:

public class Car {
    // Attributes (Properties)
    private String make;
    private String model;
    private int year;
    private String color;

    // Constructor
    public Car(String make, String model, int year, String color) {
        this.make = make;
        this.model = model;
        this.year = year;
        this.color = color;
    }

    // Methods (Behaviors)
    public void startEngine() {
        System.out.println("The engine is starting...");
    }

    public void stopEngine() {
        System.out.println("The engine is stopping...");
    }

    // Getter and Setter Methods
    public String getMake() {
        return make;
    }

    public void setMake(String make) {
        this.make = make;
    }

    // Other getters and setters...
}

B. Creating Objects from the Class:

public class Main {
    public static void main(String[] args) {
        // Creating a new Car object
        Car myCar = new Car("Toyota", "Corolla", 2020, "Red");

        // Using the Car object
        myCar.startEngine();
        System.out.println("Car make: " + myCar.getMake());
    }
}

C. Demonstrating Inheritance:

public class ElectricCar extends Car {
    private int batteryLife;

    // Constructor
    public ElectricCar(String make, String model, int year, String color, int batteryLife) {
        super(make, model, year, color); // Calling the constructor of the superclass
        this.batteryLife = batteryLife;
    }

    // New method specific to ElectricCar
    public void chargeBattery() {
        System.out.println("The battery is charging...");
    }
}

D. Polymorphism Example:

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Toyota", "Corolla", 2020, "Red");
        Car myElectricCar = new ElectricCar("Tesla", "Model S", 2022, "Blue", 300);

        myCar.startEngine();
        myElectricCar.startEngine(); // This calls the startEngine method from Car class
    }
}

4. Practical Application and Exercises (30 minutes)

A. Exercise 1: Create Your Own Class

  • Have students create a new class for another type of object, like Bicycle, with attributes and methods.

B. Exercise 2: Extend the Car Class

  • Have students create a subclass of Car, like SportsCar, and add new attributes and methods.

C. Exercise 3: Polymorphism Practice

  • Have students create an array of Car objects, including both Car and ElectricCar instances, and iterate through the array to call methods on each object.
